# 基本情報技術者試験 クイズアプリ

基本情報技術者試験の学習用4択クイズアプリです。React で構築されており、モダンなガラスモーフィズムデザインを採用しています。

## ✨ 主な機能

### 🎨 デザイン
- **白・黒・ガラスモーフィズムデザイン**: シンプルでモダンなUI
- **ダークテーマ**: 長時間の学習でも目に優しい

### 📊 学習機能
- **リアルタイム正答率表示**: 解答中の正答率をリアルタイムで確認
- **自動スクロール**: 解答後、自動で解説と次の問題ボタンへスクロール
- **間違えた問題の復習**: 間違えた問題を自動記録し、復習モードで再学習
- **ランダム出題**: 全カテゴリーから

ランダムに問題を出題
- **カテゴリー別学習**: テクノロジー、マネジメント、ストラテジーから選択

### 💡 サポート機能
- **ヒント機能**: 各問題にヒントが付き、開閉可能（初期状態は閉じている）
- **詳しい解説**: 正解・不正解に関わらず、詳しい解説を表示
- **進捗バー**: 現在の進捗を視覚的に表示

## 🚀 インストールと起動

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start
```

アプリは `http://localhost:3000` で起動します。

## 🎯 学習モード

### 1. カテゴリー別学習
- テクノロジー、マネジメント、ストラテジー、または全カテゴリーから選択
- 各カテゴリーの問題数を確認できます

### 2. ランダム出題
- 全カテゴリーからランダムに問題を出題
- 実際の試験に近い形式で学習できます

### 3. 間違えた問題のみ復習
- 間違えた問題を自動的に記録
- 苦手な分野を集中的に学習できます

## 🎨 デザインの特徴

### ガラスモーフィズム
- `backdrop-filter: blur()` を使用した透明感のあるデザイン
- 白と黒を基調としたシンプルなカラーパレット
- 繊細な境界線とシャドウ

### アニメーション
- スムーズなトランジション
- ホバーエフェクト
- 自動スクロール

## 📖 使い方

1. **モード選択**: トップページで学習モードを選択
2. **問題に回答**: 4つの選択肢から正解を選択
3. **ヒント**: 必要に応じてヒントボタンをクリック
4. **自動スクロール**: 回答後、自動で解説部分にスクロール
5. **解説確認**: 詳しい解説を確認
6. **次の問題**: 「次の問題へ」ボタンで進む
7. **正答率確認**: リアルタイムで正答率を確認
8. **結果確認**: すべて回答後、スコアと復習画面を表示

## 📊 問題データ形式

各問題は以下の形式で定義されています：

```json
{
  "id": "tech_001",
  "category": "テクノロジー",
  "subcategory": "基礎理論",
  "question": "問題文",
  "choices": ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
  "correctAnswer": 1,
  "hint": "ヒント文",
  "explanation": "解説文"
}
```

## 🎓 学習のポイント

- 各問題には、基本情報技術者試験のシラバスに基づいた内容が含まれています
- ヒントと解説をしっかり読んで、理解を深めましょう
- 間違えた問題は、復習モードで集中的に学習できます
- リアルタイム正答率で、自分の理解度を確認しながら進められます
- ランダムモードで、実際の試験に近い形式で練習できます
